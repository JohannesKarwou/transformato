utils.postprocessing(name: str = 'methane', engine: str = 'openMM', max_snapshots: int = 300, num_proc: int = 4, show_summary: bool = False, different_path_for_dcd: str = '', only_single_state: str = '', analyze_traj_with: str = 'mdtraj')

Performs postprocessing using either openMM or CHARMM and calculates the free energy estimate using MBAR.

Args:

configuration (dict): Configuration file. name (str, optional): The name of the system as deposited in the configuration file. Defaults to “methane”. engine (str, optional): The MD engine, either openMM or CHARMM. Defaults to “openMM”. max_snapshots (int, optional): Maximum number of snapshots per lambda window. Defaults to 300. show_summary (bool, optional): Plot the accumulated free energy estimate and overlap plot. Defaults to False. different_path_for_dcd (str, optional): For debugging purpose only. Defaults to “”. only_single_state (str, optional): For debugging purpose only. Defaults to “”. analyze_traj_with (str,optional): Select package for loading trajectories (either mdanalysis or mdtraj). Defaults to mdtraj.
